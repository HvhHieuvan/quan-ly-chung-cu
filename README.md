# Hệ thống quản lý chung cư

## 1. Giới thiệu đề tài
Dự án xây dựng hệ thống quản lý chung cư phục vụ cho 3 vai trò:
- Cư dân
- Ban quản lý
- Admin

Hệ thống hỗ trợ quản lý căn hộ, cư dân, hóa đơn, phản ánh, thẻ xe và thông báo.

## 3. Giảng viên hướng dẫn
- Võ Việt Khoa

## 4. Công nghệ sử dụng
- Backend: Spring Boot
- Frontend: ReactJS
- Database: MySQL
- Security: Spring Security + JWT
- Quản lý mã nguồn: GitHub

## 5. Cấu trúc dự án
- docs/: tài liệu phân tích và thiết kế
- weekly-reports/: báo cáo tiến độ theo tuần
- backend/: mã nguồn backend
- frontend/: mã nguồn frontend
- database/: script cơ sở dữ liệu
